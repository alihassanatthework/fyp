================================================================
 ME — AI Skin & Scalp Assistant  ·  SETUP INSTRUCTIONS
 (Portable transfer package — full environment included)
================================================================

This zip contains the COMPLETE project: source code, trained model
weights, the SQLite database with current data, and the .env file with
real secrets (Groq, email, Safepay). It is for PRIVATE transfer to your
own device only — do not share it.

----------------------------------------------------------------
 0. REQUIRED VERSIONS
----------------------------------------------------------------
  - Python : 3.9.x   (developed on 3.9.6)
  - Node   : 18+      (developed on Node 26, npm 11)  -- Node 18 or 20 also fine
  - OS     : macOS / Linux / Windows
  - Disk   : ~3 GB (model weights are large)
  - RAM    : 4 GB+ recommended (ML models load into memory)

----------------------------------------------------------------
 1. UNZIP
----------------------------------------------------------------
  Unzip anywhere, e.g.:
      ~/Desktop/ME_project/
  Open a terminal in that folder (the one containing manage.py).

----------------------------------------------------------------
 2. BACKEND (Django) — Python setup
----------------------------------------------------------------
  Create a fresh virtual environment (the old venv was NOT included):

      python3 -m venv venv
      source venv/bin/activate           # Windows: venv\Scripts\activate

  Install dependencies:

      pip install --upgrade pip
      pip install -r requirements.txt

  Apply database migrations (DB is already included with data, but this
  ensures the schema is in sync — it is safe and non-destructive):

      python manage.py migrate

  (Optional) create an admin user if you need the Django admin:

      python manage.py createsuperuser

----------------------------------------------------------------
 3. FRONTEND (React) — Node setup
----------------------------------------------------------------
  node_modules was NOT included — reinstall it:

      cd react
      npm install

  Then either build for production OR run the dev server:

      npm run build         # production build served by Django
        -- OR --
      npm start             # dev server at http://localhost:3000
      npm run start:https   # dev server over HTTPS (needed for camera on phone/LAN)

  cd ..

----------------------------------------------------------------
 4. RUN THE BACKEND
----------------------------------------------------------------
      source venv/bin/activate            # if not already active
      python manage.py runserver

  Backend:  http://localhost:8000
  Frontend (dev): http://localhost:3000   (proxies /api to :8000)

----------------------------------------------------------------
 5. .ENV — ALREADY CONFIGURED
----------------------------------------------------------------
  The .env file is INCLUDED with real values, so no setup is needed:
    - GROQ_API_KEY        (LLM recommendations — Groq llama-3.3-70b)
    - EMAIL_HOST_USER / EMAIL_HOST_PASSWORD  (report + password-reset email)
    - SAFEPAY_*           (sandbox payment gateway: cards + Easypaisa + JazzCash)

  Models are included under:
    - core/ai_models/*.pth        (skin, scalp, dryness, normal classifiers)
    - core/models/yolo_scalp.pt   (legacy scalp YOLO fallback)
    - core/models/unet_checkpoints/  (U-Net eczema segmentation)

  Database included: db.sqlite3 (current data).

----------------------------------------------------------------
 6. QUICK START (copy-paste)
----------------------------------------------------------------
      python3 -m venv venv && source venv/bin/activate
      pip install -r requirements.txt
      python manage.py migrate
      cd react && npm install && npm run build && cd ..
      python manage.py runserver
      # open http://localhost:8000

----------------------------------------------------------------
 NOTES
----------------------------------------------------------------
  - The dataset/ training folder was intentionally EXCLUDED (not needed to run).
  - To retrain the scalp model: bash scripts/train_scalp.sh (needs the dataset).
  - Camera capture needs HTTPS or localhost (use npm run start:https on a phone).
  - LLM: Groq (primary, llama-3.3-70b) with Ollama (local) fallback.
